# Worm Arena

เกมหนอนแบบเล่นบนเว็บ เปิดลิงก์แล้วเล่นได้ทั้งบนคอมและมือถือ

## เปิดในเครื่อง

เปิดไฟล์ `index.html` ด้วย Live Server หรือเบราว์เซอร์ได้ทันที

## เอาขึ้นเว็บแบบมีลิงก์

โปรเจ็กต์นี้ถูกเตรียมไว้สำหรับ GitHub Pages แล้ว

1. สร้าง repository ใหม่บน GitHub
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น branch `main`
3. เปิด GitHub repository ของคุณ
4. ไปที่ `Settings` > `Pages`
5. ที่หัวข้อ `Build and deployment` เลือก `Source` เป็น `GitHub Actions`
6. push โค้ดอีกครั้งหรือรอ workflow `Deploy Static Site` ทำงาน
7. หลัง deploy เสร็จ GitHub จะให้ลิงก์เว็บสาธารณะสำหรับเปิดเล่นได้เลย

## มือถือ

เปิดลิงก์ GitHub Pages จากมือถือได้ทันที และตัวเกมมีปุ่มสัมผัสสำหรับควบคุมแล้ว
